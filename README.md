# tad-rust

A project for learning Abstract Data Types (ADTs) in Rust.
Heavily inspired by Java's implementations, so don't expect idiomatic Rust.

## Structures
- [x] Linked List
- [x] Stack
- [x] Queue
- [x] Heap
- [x] Binary Tree
- [x] HashMap
- [ ] Graph
- [ ] Binary Search Tree (BST)
- [ ] Prefix Tree

## Features
- [x] Debug
- [ ] Custom errors with thiserror
- [ ] IntoIterator (for)
- [ ] FromIterator (for with collect)
- [x] Add get_mut/peek_mut to the structures
- [ ] Make a stack, linkedlist and queue from a Vec! with ::from
- [ ] Documentation
- [ ] Example folder with demo
